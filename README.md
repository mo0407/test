# SBTI 人格测试

一个基于 HTML/CSS/JavaScript 的单页面人格测试应用。

## ✨ 特性

- 🎯 30道精心设计的问题
- 🧠 15个维度分析系统
- 🎭 27种独特人格类型
- 📱 完全响应式设计
- ⚡ 单文件部署，无需后端
- 🎨 现代化 UI 设计

## 🎯 功能说明

### 测试流程
1. 用户点击"开始测试"
2. 完成30道选择题
3. 系统自动分析15个维度得分
4. 匹配最佳人格类型并显示结果

### 维度系统
- **自我模型**: S1-S3 (自我认知、自我清晰、成长动力)
- **情绪模型**: E1-E3 (安全感、投入度、独立性)
- **态度模型**: A1-A3 (信任倾向、规则意识、目标感)
- **行动模型**: Ac1-Ac3 (成就动机、决断力、执行力)
- **社交模型**: So1-So3 (社交主动性、人际边界感、表达与真实度)

### 人格类型示例
- **CTRL 拿捏者**: 宇宙熵增定律的反抗者
- **BOSS 领导者**: 永恒向上定律的拥有者
- **HHHH 傻乐者**: 思维回路清奇的独特人格
- ... 以及其他24种人格类型

## 🔧 自定义

### 修改题目
编辑 `sbti_app.html` 中的 `questions` 数组

### 添加人格类型
编辑 `TYPE_LIBRARY` 对象和 `TYPE_IMAGES` 映射

### 调整样式
修改 `<style>` 标签内的 CSS 变量
